STEP 1: START SERVER

bitcoind -regtest -daemon

STEP 2: CREATE PRIVATE BLOCKCHAIN (No need to do it every time)

bitcoin-cli -regtest setgenerate true 101

STEP 3: CREATE PUBLIC KEY

bitcoin-cli -regtest getnewaddress

STEP 4: SEND COINS TO THAT ADDRESS FROM BLOCKCHAIN

bitcoin-cli -regtest sendtoaddress $NEW_ADDRESS 10.00

returns: txID in hexadecimal

STEP 5: SEE CONFIRMED AND UNCONFIRMED TRANSACTIONS

bitcoin-cli -regtest listunspent 0 

STEP 6: CONFIRM RECENT TRANSACTION

bitcoin-cli -regtest setgenerate true 1

STEP 7: DISPLAY CONFIRMED TRANSACTIONS

bitcoin-cli -regtest listunspent

get txID and vOUT out of the output

STEP 8: GENERATE A RECEIVER ADDRESS

mz6KvC4aoUeo6wSxtiVQTo7FDwPnkp6URG
